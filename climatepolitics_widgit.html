iframe

The current state of a widgit is an iframe. Domain hosts may add the line: iframe src="./iframe_test_files/Bill_Dodd_(USA_CA).html" name="test" height="420" width="700">

This iframe is a rendering of our MW page that omits nonessentials such as the sidebar. If we want to move forward with this widgit:

1. We need to make the omition of nonessentials tenacious, currently navigating off of the intial page of the widgit will render full MW pages.

2. The wigit ought to include the navigation feature of a "back button".

3. The widgit (in most cases) ought to start with a navigation page (opposed to the profile of a spcific candidate as is demonstrated).
