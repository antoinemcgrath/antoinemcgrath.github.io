iframe

The current state of a widget is an iframe. Domain hosts may add the line: iframe src="http://www.climatepolitics.info/wiki/Bill_Dodd_(USA_CA)?action=render&title=Main_Page" name="test" height="420" width="700">

This iframe is a rendering of our MW page that omits nonessentials such as the sidebar. If we want to move forward with this widget:

1. We need to make the omition of nonessentials tenacious, currently navigating off of the initial page of the widget will render full pages rather than reduced versions.

2. The widget ought to include the navigation feature of a "back button".

3. The widget (in most cases) ought to start with a navigation page (opposed to the profile of a specific candidate as is demonstrated).
